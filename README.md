## Technologies Used

I built this project with a lot of "vibe coding" assistance from Gemini:

- **HTML5** & **CSS3** (Structure, layout, and smooth animations)
- **SVG** (Scalable vector graphics for articulatory organs)
- **JavaScript** (Real-time coordinate manipulation and state management)
- **Google Gemini** (AI collaboration for coding and optimization)
